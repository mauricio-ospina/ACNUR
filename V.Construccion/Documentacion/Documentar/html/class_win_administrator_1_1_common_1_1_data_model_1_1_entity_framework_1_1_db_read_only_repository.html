Detailed Description

A DbReadOnlyRepository is a IReadOnlyRepository interface implementation representing the collection of all entities in the unit of work, or that can be queried from the database, of a given type. DbReadOnlyRepository objects are created from a DbUnitOfWork using the GetReadOnlyRepository method. DbReadOnlyRepository provides only read-only operations against entities of a given type.
