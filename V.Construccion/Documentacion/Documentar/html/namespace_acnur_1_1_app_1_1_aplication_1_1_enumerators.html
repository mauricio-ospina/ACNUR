enum Acnur.App.Aplication.Enumerators.ExceptionHandlingPolitic
strong

Defines the exception handling politics to use.

Enumerator
Propagate 

Represents the propagation politic. This politic specifies that every exception will be passed to the caller without any transformation.

Wrap 

Represents the wrap politic. This politic specifies that every exception will be wrapped inside another exception and then passed to the caller.

Replace 

Represents the replacement politic. This politic specifies that every exception will be replaced for another exception; then the new exception will be passed to the caller.

The original exception will be lost.
