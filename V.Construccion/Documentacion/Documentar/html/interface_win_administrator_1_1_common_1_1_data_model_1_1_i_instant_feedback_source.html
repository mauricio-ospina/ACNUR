Detailed Description

A data source suitable as an Instant Feedback source. The GetList() method of the base IListSource interface is expected to return an instance of an internal type that happens to implement the IList interface. As such the IInstantFeedbackSource interface can only be implemented as a wrapper for an existing Instant Feedback source, e.g. EntityInstantFeedbackSource or WcfInstantFeedbackDataSource.
