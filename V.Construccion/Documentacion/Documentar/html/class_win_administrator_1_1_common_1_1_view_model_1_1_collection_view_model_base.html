Detailed Description

The base class for POCO view models exposing a collection of entities of a given type and CRUD operations against these entities. It is not recommended to inherit directly from this class. Use the CollectionViewModel class instead.

Template Parameters
TEntityA repository entity type.
TProjectionA projection entity type.
TPrimaryKeyA primary key value type.
TUnitOfWorkA unit of work type.

Parameters
unitOfWorkFactoryA factory used to create a unit of work instance.
getRepositoryFuncA function that returns a repository representing entities of the given type.
projectionA LINQ function used to customize a query for entities. The parameter, for example, can be used for sorting data and/or for projecting data to a custom type that does not match the repository entity type.
newEntityInitializerA function to initialize a new entity. This parameter is used in the detail collection view models when creating a single object view model for a new entity.
canCreateNewEntityA function that is called before an attempt to create a new entity is made. This parameter is used together with the newEntityInitializer parameter.
ignoreSelectEntityMessageA parameter used to specify whether the selected entity should be managed by PeekCollectionViewModel.
