Detailed Description

The base class for POCO view models exposing a single entity of a given type and CRUD operations against this entity. This is a partial class that provides the extension point to add custom properties, commands and override methods without modifying the auto-generated code.

TPrimaryKeyA primary key value type.
TUnitOfWorkA unit of work type.
